I'm a senior finance professional with **7+ years** across portfolio management, M&A underwriting, debt syndication, and New Markets Tax Credit (NMTC) transactions — currently focused on **community development finance** at the intersection of capital markets and mission-driven lending.

- 🔭 I build **production Python tooling** for CDFIs, MDIs, and fair-lending practitioners — **22 packages published on PyPI**
- 🧮 Day-to-day: credit analysis, structured-finance modeling, and turning messy regulatory data (FDIC, CDFI Fund, HMDA, FFIEC) into decisions

## 🚀 Featured Work

A portfolio of **open-source Python packages** for community development finance — each CI-published to PyPI with automated tests. Full catalog: **[pypi.org/user/thejaypatel1511](https://pypi.org/user/thejaypatel1511/)**

| Project | What it does |
|---|---|
| **[nmtc-application-builder](https://pypi.org/project/nmtc-application-builder/)** | Flagship NMTC application-intelligence platform — pipeline analysis, eligibility validation, readiness scoring, and visualization for CDEs |
| **[cdfi-benchmark](https://pypi.org/project/cdfi-benchmark/)** | CDFI &amp; MDI peer benchmarking on FDIC call-report data — NIM, efficiency ratio, ROAA, capital ratios |
| **[fair-lending-screener](https://pypi.org/project/fair-lending-screener/)** | Statistical disparate-impact analysis for HMDA data — the methodology federal examiners use, open-sourced for advocates |
| **[nmtc-mapper](https://pypi.org/project/nmtc-mapper/)** | Automated NMTC eligibility checker — geocodes addresses and checks Low-Income Community status via CDFI Fund + Census data |
| **[cdfi-val](https://github.com/Jaypatel1511/cdfi-val)** | Valuation toolkit for CDFIs and Minority Depository Institutions — TBV, P/E, comps, DCF |
| **[waterfall-py](https://github.com/Jaypatel1511/waterfall-py)** | Debt-waterfall engine for structured finance — senior/mezz/equity distribution, DSCR tracking, cash-sweep logic |
